## 1) 로그인과 대시보드
- 접속 주소: `https://startup-sejong.vercel.app/admin/`
- 아이디/비밀번호로 로그인합니다.
- 대시보드에서는 다음을 한눈에 볼 수 있습니다.
  - 전체 공지 개수, 고정된 공지 개수
  - 전체 일정, 예정된 일정 개수
  - 최근 등록된 공지 목록
  - 빠른 작업: 새 공지 만들기, 새 일정 추가

## 2) 공지사항 관리
- 경로: `https://startup-sejong.vercel.app/admin/announcements`
- 할 수 있는 일
  - 새 공지 만들기, 수정하기, 삭제하기
  - 상단에 고정하기(자주 봐야 하는 공지에 사용)
  - 첨부파일 추가/삭제/다운로드
- 빠르게 작성하는 방법
  1) “새 공지사항 작성” 클릭
  2) 제목 입력 → 카테고리 선택(중요 공지는 ‘중요’), 필요한 경우 ‘상단 고정’ 체크
  3) 내용 입력: 편집기(에디터)에 바로 작성 → 상단 ‘미리보기’로 결과 확인 가능
  4) 파일 첨부(선택): 최대 5개까지 추가 가능
  5) 알림도 함께 보내려면 ‘푸시 알림’ 체크
  6) 저장 버튼 클릭 → 완료 알림 확인
  7) 목록에서 방금 작성한 공지 확인

- 작성 팁
  - 제목은 간단명료하게, 내용은 핵심부터 적습니다.
  - 중요한 안내는 카테고리 ‘중요’ + ‘상단 고정’을 함께 사용하면 더 눈에 띕니다.
  - 파일 이름은 내용을 알기 쉽게 적어주세요(예: 신청서_2025학년도.pdf).

## 3) 학사일정 관리
- 경로: `https://startup-sejong.vercel.app/admin/calendar`
- 할 수 있는 일
  - 일정 만들기, 수정하기, 삭제하기
  - 일정 유형 선택(학기/시험/휴일/신청/기타)
  - 중요 일정 표시(중요 표시하면 리스트에서 눈에 잘 띕니다)
- 작성 팁
  - 일정 제목은 “무엇을, 언제”가 보이게 적습니다.
  - 설명에 필요한 링크나 장소가 있으면 함께 적습니다.

## 4) 알림 보내기(푸시 알림)
- 경로: `https://startup-sejong.vercel.app/admin/notifications`
- 이 화면에서 하는 일
  - 현재 알림을 받아보기로 한 인원(활성 구독자 수)을 확인
  - 최근 보낸 알림의 성공/실패 기록 확인
  - 알림 내용 작성 후 전체에게 발송
- 보내는 방법(간단 4단계)
  1. 상단 카드에서 활성 구독자 수를 확인합니다(0명이면 발송해도 받는 사람이 없습니다).
  2. 공지/일정 중에서 보낼 내용을 선택하거나, 직접 제목·내용을 입력합니다.
  3. 사용자가 눌렀을 때 이동할 화면(예: 공지사항, 학사일정)을 선택합니다.
  4. “발송”을 누르고 결과(성공/실패 건수)를 확인합니다.
 - 참고
   - 공지 작성 화면에서도 ‘푸시 알림’ 체크 후 저장하면 즉시 발송됩니다. 별도로 이 화면에 들어오지 않아도 됩니다.
   - 이미 보낸 내용을 다시 안내하거나, 여러 내용을 모아 보내고 싶을 때는 이 화면을 사용하세요.
- 구독자 관리(선택 사항)
  - “구독자 보기”를 누르면 알림을 받는 기기 목록이 나옵니다.
  - 오래되었거나 더 이상 사용하지 않는 기기는 “비활성화/삭제”할 수 있습니다.
  - “관리 도구”에서 비활성 구독 정리, 오래된 구독 정리 버튼으로 깔끔하게 정리할 수 있습니다.
- 발송 팁
  - 제목은 짧고 명확하게(최대 한 줄), 내용은 핵심만 1~2문장.
  - 너무 자주 보내지 마세요. 중요한 내용 위주로 보내면 효과가 큽니다.
  - 시험/신청처럼 꼭 봐야 하는 내용은 “중요 알림” 옵션을 사용하세요.

## 5) 알림을 받으려면(사용자 안내용)
- 알림 받기 버튼 누르기
  - 홈페이지 상단의 “알림 받기”를 누르고 휴대폰/브라우저에서 “허용”을 선택해야 합니다.
- 아이폰 사용자의 경우
  - Safari로 사이트를 열고, ‘공유’ 버튼 → “홈 화면에 추가”로 설치한 뒤에 알림이 동작합니다.
  - 아이폰의 Chrome 등 다른 브라우저에서는 알림이 동작하지 않을 수 있습니다.
- 안드로이드/데스크톱
  - Chrome 등에서 “알림 허용”을 하면 정상적으로 알림을 받습니다.
- 알림이 안 온다고 하면 이렇게 안내하세요
  - “알림 받기”를 다시 눌러 권한을 허용했는지 확인
  - 휴대폰 설정 → 해당 브라우저의 알림이 꺼져있지 않은지 확인
  - 아이폰이면 Safari로 접속 후 “홈 화면에 추가”했는지 확인

## 6) 자주 묻는 질문(FAQ)
- Q. 발송했는데 받는 사람이 없다고 나옵니다.
  - A. 아직 “알림 받기”를 누른 사람이 없을 수 있습니다. 먼저 학생들에게 알림 받기 방법을 안내해주세요.
- Q. 아이폰에서 알림이 안 와요.
  - A. Safari로 접속해 “홈 화면에 추가”한 뒤에 가능합니다. Chrome 등 다른 브라우저는 아이폰에서 알림이 제한됩니다.
- Q. 첨부파일이 안 열려요.
  - A. 파일을 다시 올려보거나, 파일 형식을 바꿔서 올려주세요(예: PDF). 파일 이름도 명확하게 작성해주세요.
- Q. 같은 알림을 여러 번 보내도 될까요?
  - A. 너무 자주는 피해주세요. 중요한 일정이나 마감 공지 위주로, 필요한 시점에만 보내는 것이 좋습니다.

# 부록 — 화면 캡처 포함 따라하기 예시

아래는 관리자 입장에서 자주 하는 작업을 순서대로 따라할 수 있도록 정리한 예시입니다. 스크린샷 파일은 추후 `img/admin/` 폴더에 추가될 예정입니다(표시: [캡처]).

## A. 새 공지 만들기 + (선택) 알림 함께 보내기
1) 관리자 접속: `/admin` → 상단 “빠른 작업”에서 “새 공지사항 작성” 클릭 [캡처]
2) 제목 입력: 예) `[중요] 2025학년도 1학기 융합창업 연계전공 신청 안내` [캡처]
3) 분류/고정: 카테고리에서 ‘중요’ 선택, 필요하면 ‘상단 고정’ 체크 [캡처]
4) 내용 입력: 편집기(에디터)에 작성 → ‘미리보기’로 결과 확인 [캡처]
5) 첨부파일 추가(선택): ‘파일 첨부’에서 파일 선택 → 목록 확인 [캡처]
6) 알림 함께 보내기(선택): ‘푸시 알림’ 체크(체크하지 않으면 알림 없이 공지만 등록) [캡처]
7) 저장: 완료 알림 확인 후 목록으로 이동 [캡처]

추천 문구 예시
- 제목: `[공지] 1학기 전공 신청 안내`
- 내용: `신청 기간 3/1~3/10, 자세한 내용은 공지에서 확인하세요.`

체크 포인트
- 제목은 한 줄, 내용은 핵심 1~2문장으로 간단히
- 너무 잦은 발송은 피하고, 중요한 시점에만

## B. 새 학사일정 추가
1) 경로: `/admin/calendar` → “새 일정 추가” 클릭 [캡처]
2) 일정 제목: 예) `중간고사 기간`
3) 일정 기간: 시작/종료 날짜 선택
4) 유형: `시험` 선택(학기/시험/휴일/신청/기타 중 선택)
5) 설명: 간단한 안내 문구(필요한 경우 장소/링크)
6) 중요 표시: 일정이 특히 중요하면 체크
7) 저장 클릭 → 목록에서 반영 확인 [캡처]

(선택) 일정 알림 보내기
- 푸시 알림 관리 → ‘예정된 일정’에서 방금 만든 일정 선택 → 제목/내용 자동 채움 확인 → 발송

## C. 학생에게 “알림 받기” 안내하기
안내 문구 예시(복사해 공지에 붙여 쓰기 좋습니다)

안내 1: 공통
- 홈페이지 상단의 ‘알림 받기’ 버튼을 눌러 주세요.
- 브라우저에서 “알림 허용”을 선택해야 알림이 도착합니다.

안내 2: 아이폰(iPhone)
- Safari로 접속해 ‘공유’ 버튼 → “홈 화면에 추가”로 설치한 뒤 알림이 동작합니다.
- 아이폰의 Chrome 등 다른 브라우저에서는 알림이 제한될 수 있습니다.

안내 3: 안드로이드/데스크톱
- Chrome 등에서 “알림 허용”을 하면 정상적으로 알림을 받습니다.

문제 해결(학생 기준)
- 알림이 안 오면: 알림 허용 다시 시도, 해당 브라우저의 알림이 휴대폰 설정에서 꺼져 있지 않은지 확인, 아이폰은 Safari + 홈 화면 추가 확인

## D. 자주 쓰는 템플릿 묶음
- 공지 등록 알림
  - 제목: `[공지] 새 공지 등록`
  - 내용: `중요한 안내가 올라왔습니다. 자세한 내용은 공지에서 확인하세요.`
  - 이동: `공지사항`

- 일정 임박 알림
  - 제목: `[일정] 내일 마감 안내`
  - 내용: `마감이 임박했습니다. 자세한 일정은 학사일정에서 확인하세요.`
  - 이동: `학사일정`

- 긴급 안내 알림(중요 알림 권장)
  - 제목: `[긴급] 즉시 확인 바랍니다`
  - 내용: `중요 공지가 등록되었습니다. 꼭 확인해 주세요.`
  - 이동: `공지사항` / 옵션: ‘중요 알림’ 체크
